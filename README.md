# Amazon-India-Sales-Dashboard---2025
•	Developed an interactive end-to-end sales dashboard to track key performance indicators (KPIs) for Amazon India, processing data for 1.12Bn in revenue and 15K total orders.
•	Designed high-level KPI cards to monitor Total Profit ($895M), Customer Acquisition (7K), and average order values to provide leadership with an immediate health check of the business.
•	Visualized sales distribution across categories (Beauty, Electronics, Books, etc.), identifying Beauty as the top-performing segment and analysing the quantity-to-revenue ratio per category.
•	Developed a Time-Series Line Chart to track monthly fluctuations, identifying a peak sales period in December ($97.5M) and a significant dip in February.
•	Analysed payment preferences using a Donut Chart, revealing a near-even split between Credit Card, Debit Card, UPI, and COD, highlighting the need for versatile payment gateway support.
